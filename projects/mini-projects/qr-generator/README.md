# QR Code Generator 🔳

A simple and user-friendly QR Code Generator built with Python using the qrcode library.

This project allows users to generate different types of QR codes directly from the terminal.

---

## 🚀 Features

* 📝 Text to QR Code
* 🌐 URL to QR Code
* 💳 Payment QR Code (UPI Supported)
* 📁 Save QR as PNG Image
* 📋 Menu-driven Interface
* ✅ Simple and Beginner Friendly

---

## 📷 QR Types Supported

### 1. Text QR Code

Convert any text or message into a QR code.

### 2. URL QR Code

Generate QR for websites or links.

Example:

```text
https://google.com
```

### 3. Payment QR Code

Create payment QR using UPI format.

Example:

```text
upi://pay?pa=yourupi@upi&pn=YourName&am=100
```
