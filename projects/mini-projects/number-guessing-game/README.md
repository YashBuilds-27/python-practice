# 🎯 Number Guessing Game

A fun and simple Python terminal game where the computer generates a random number between **1 and 100**, and you have **5 attempts** to guess it correctly.

---

## 🚀 Features

- 🎲 Random number generation
- 🔥 Hint system based on closeness
- 🔁 Replay option
- ⚠️ Error handling for invalid input
- 🎯 Limited attempts for challenge

---

## 🕹️ How to Play

1. Run the Python file.
2. Press any key to start.
3. Guess the hidden number between **1 to 100**.
4. You get hints after every wrong guess:

- 🔥 Very close
- 😊 Close
- 😐 Far
- 🥶 Very Far

5. Guess within **5 attempts** to win.

---

## 💻 Code Used

- `random` module
- `while loop`
- `try-except`
- `if-elif-else`
- `abs()` for distance checking
