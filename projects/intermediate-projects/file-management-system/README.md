# 📁 Python File Management System

A menu-driven **File Management System** built using **Python** and the **os module**.

This project allows users to create, read, write, append, and delete files directly from the terminal. It is a practical beginner project that demonstrates core Python programming concepts and real-world file operations.

---

## ✨ Features

* ✅ Create New File
* ✅ Read File Content
* ✅ Write Data to File
* ✅ Overwrite Existing Content with Confirmation
* ✅ Append Data at Specific Line Number
* ✅ Append Data at Specific Index Position
* ✅ Delete File
* ✅ Input Validation
* ✅ Confirmation Before Sensitive Actions

---

## 🛠️ Technologies Used

* Python 3
* os Module
* File Handling

---

## 📌 Main Menu

```text
1. Create a File
2. Read File
3. Write in a File
4. Append a File
5. Delete a File
6. Exit
```
