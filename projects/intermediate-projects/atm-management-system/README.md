# 🏧 Python ATM Management System

**ATM Management System** built using **Python**, **JSON**, and **File Handling**.
This project was created as a learning project to practice programming fundamentals and real-world logic building.

---

## ✨ Features

* ✅ Create New Account
* ✅ Login with Account Number & PIN
* ✅ Check Balance
* ✅ Deposit Money
* ✅ Withdraw Money
* ✅ View Transaction History
* ✅ Data stored using JSON
* ✅ Separate transaction log for each account

---

## 🛠️ Technologies Used

* Python 3
* JSON
* File Handling
* Datetime Module
