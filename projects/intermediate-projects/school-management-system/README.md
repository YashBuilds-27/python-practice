# 🏫 School Management System

A console-based **School Management System** built using **Python** and **MySQL**.
This project manages student records, teacher details, attendance, fee structure, and library data using a database-driven approach.

Designed as an academic project to practice **Python + SQL integration**.

---

## ✨ Features

### 👨‍🎓 Student Management

* Add Student
* Remove Student
* Display Student Details

### 👨‍🏫 Teacher Management

* Add Teacher
* Remove Teacher
* Update Salary
* Display Teacher Details

### 📅 Attendance Management

* Class Attendance
* Teacher Attendance
* View Attendance Records

### 💰 Fee Management

* Update Fee Structure
* Display Fees

### 📚 Library Management

* Add Book
* Remove Book
* Display Books
